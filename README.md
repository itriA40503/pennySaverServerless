# PennySaver Serverless #

## What is this repository for? ##
* Control QR code link to some where.
* Link to App Store or Google play.

## How do I get set up? ##
```
npm install serverless
```
* for install serverless framework

```
serverless deploy -v
```
* for deploy your service
